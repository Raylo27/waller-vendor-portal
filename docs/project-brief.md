# City of Waller Vendor Portal
## Project Brief

### What this is
A lightweight pilot and demonstration portal for the City of Waller that shows what a modern vendor portal could look like.

### Why it matters
It gives local businesses a clear way to register, browse opportunities, and submit proposals digitally while giving city staff a simpler way to post opportunities and review submissions.

### Who it serves
- Vendors
- City staff

### Core promise
A staff user can post an opportunity, and vendors can see it right away.

### MVP scope
Vendor registration, opportunity browsing, opportunity detail, submission flow, staff login, staff dashboard, opportunity posting, vendor registry, and submissions review.

### What is simplified in the demo
- File uploads store filenames only
- Dashboard metrics can be hardcoded
- Demo authentication is simplified
- Vendor registry can be seeded/static
- Branding uses city-branded header language unless permission is granted for official seal usage
