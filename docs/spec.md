# Product Spec

## Project
City of Waller Vendor Portal

## Status
Approved revised spec is the current source of truth.

## Positioning
This MVP is a lightweight pilot and demonstration layer designed to show what a modern vendor portal could look like for the City of Waller. It is not a claim to replace the city's full procurement process immediately.

## Core User Flows
- Vendor registration
- Opportunity browsing
- Opportunity detail view
- Proposal submission
- Staff login
- Staff dashboard
- Post new opportunity
- Vendor registry
- Submissions review

## Core Demo Moment
A staff user posts an opportunity and it appears immediately on the vendor-facing opportunities list.
