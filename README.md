
# City of Waller Vendor Portal

A lightweight pilot and demonstration portal designed to show what a modern vendor portal could look like for the City of Waller.

## Status
Soft planning phase complete. Build phase not started yet.

## Purpose
This project is intended to demonstrate a simple public-facing vendor portal where businesses can:
- register as vendors
- browse city opportunities
- submit proposals digitally

It also includes a staff-side experience where city staff can:
- post opportunities
- view registered vendors
- review submissions

## MVP Goal
The MVP is a pilot-layer demo, not a full replacement for the city's procurement process.

## Core Demo Moment
A staff user posts a new opportunity, and it appears immediately on the vendor-facing opportunities page.

## Planned Stack
- Frontend: React + Vite
- Backend: Express
- Database: PostgreSQL
- ORM: Prisma

## Current Priority
Complete planning, positioning, messaging, and project structure before coding begins.

## Notes
Branding, security hardening, and production-grade workflows are deferred until after demo validation.
